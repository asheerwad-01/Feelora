# 🎵 Feelora

> **Don't listen to music. Step inside it.**

Feelora is an immersive music experience that transforms your Spotify library into a living 3D universe. Instead of scrolling through playlists, users can explore their music from within a dynamic sphere of songs, synchronized lyrics, and emotion-driven visuals.

---

## ✨ Features

### 🎧 Spotify Integration

Connect your Spotify account and access:

* Liked Songs
* Playlists
* Recently Played Tracks
* Saved Albums
* Listening History

### 🌌 Immersive 3D Music Universe

Experience your music in an entirely new way.

* Interactive song sphere
* Spatial navigation
* Dynamic visual effects
* Audio-reactive animations
* Smooth camera movement

### 🎤 Synchronized Lyrics

Follow lyrics in perfect sync with the currently playing track.

* Real-time lyric highlighting
* Smooth lyric transitions
* Immersive reading experience

### ⚡ Premium User Experience

Designed to feel fluid, cinematic, and modern.

* GSAP-powered animations
* Smooth scrolling
* Glassmorphism effects
* Responsive layouts
* Modern typography

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/feelora.git

cd feelora
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

## 🎯 Vision

Most music platforms focus on organization.

Feelora focuses on experience.

Music is emotional, personal, and deeply connected to memory. Feelora transforms playlists into interactive worlds, allowing users to explore their music visually instead of browsing endless lists.

Our goal is to create the future of music interaction—where songs become places, lyrics become experiences, and listening becomes exploration.
